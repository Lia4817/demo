### 拉格朗日插值

> n次插值条件：n+1个不重合节点 
> 
> Ln(x)=Σyk*lk(x)    
> 
> 基函数：lk(x)去掉本身零点作为分子，xk代入本身作为分母

> n = 1 两个点 一次插值（线性插值）
>
> n = 2 三个点 二次插值（抛物线插值）
